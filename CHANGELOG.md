# 更新日志 (Changelog)

所有重要的项目变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-01-24

### 🌍 新增功能
- **多语言支持**: 新增14种语言支持，覆盖Upwork全球80-90%自由职业者
  - 第一优先级: 西班牙语(es)、印地语(hi)、葡萄牙语(pt)、孟加拉语(bn)
  - 第二优先级: 乌克兰语(uk)、菲律宾语(tl)、俄语(ru)、阿拉伯语(ar)
  - 第三优先级: 日语(ja)、韩语(ko)、德语(de)、法语(fr)
- **智能紧凑显示**: 为长单词语言提供简洁版文本，避免布局问题
- **语言列表优化**: 英文调整为第一选项，中文第二选项

### 🏗️ 架构变更
- **i18n 模块重构**: 将国际化代码从主文件中独立出来
- 新增 `contents/i18n/` 目录结构，包含17个语言相关文件
- 完整的 TypeScript 类型定义和接口
- 模块化设计，提高代码可维护性
- 主文件 `upwork-monitor.ts` 从773行优化到759行

### 📱 界面优化
- **紧凑显示模式**:
  - 英文: "Proposals" → "Prop", "Total" → "Total"
  - 中文: "投标情况" → "投标", "面试中" → "面试"
  - 德语: "Vorschläge" → "Vorsch", "Gespräch" → "Gespr"
  - 法语: "Propositions" → "Prop", "Entretien" → "Entre"
- 空间利用率提升30-60%，特别优化德语、法语等长单词语言
- 智能文本回退机制：未定义简短版本的语言自动使用完整版本

### 🐛 错误修复
- 修复首次安装时默认语言设置问题（现在默认使用英文）
- 修复 popup 语言选择器的兼容性问题
- 修复类型安全相关的 TypeScript 错误

### 📦 项目信息
- 版本号从 0.0.1 升级到 1.0.0
- 项目描述更新，突出多语言支持特性
- 构建系统优化，支持新的模块化结构
